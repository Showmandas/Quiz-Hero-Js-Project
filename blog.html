1. Difference between map,filter,forEach and find function:

map() : This function iterate on every elements of an array and return a new array.And Update the array elements based on any condition.

filter() : function iterate over an array based on some condition or filter out and return a new array.

forEach() : function iterate over an each array elements but does not change the array

find() : function return the first item or single item from an array based on some condition.

localStorage Vs sessionStorage Vs cookies:

localStorage :It has no expire date or time whenever close the browser or tab or computer.It will be saved in browser.It's storage capacity is 10mb.It stores data in browser.

sessionStorage :It has expire date or time.When the tab or browser is closed the data has been lost from the browser.It's storage capacity is 5mb.It stores data only in browser.

cookies :It's manually set on the browser.It's storage capacity is 4kb.It will be sent from the server.It stores data in both browser and server.

What is event loop and how does it work?

We know that Javascript is a synchronus and single threaded language.It executes the line of code one at a time or line by line serially.But in some scenarios it behaves as an asynchronus.If we see as an example of timer or setInterval,setTimeOut funtion,these are works as .In this situation this mechanism handle this.It manages the cycle of callback function from the queue to the call stack.
A call stack will be empty for bringing anything in it for event loop.When the call stack is empty,the first unit will be entered to the queue to bring to the call stack by the event loop
